HTML

HTML atau Hypertext Markup Languange adalah bahasa markup standar yang digunakan untuk membuat struktur dan konten halaman web. HTML berfungsi untuk membuat struktur kerangka dasar sebuah website.

CSS atau Cascading StyleSheet adalah bahasa gaya bahasa (style sheet language) yang digunakan untuk mengatur tampilan visual elemen yang ditulis dalam bahasa markup seperti HTML. Biasanya, CSS digunakan untuk mengatur estetika website, tata letak, hingga warna juga ukuran teks.
